Angular JS

¿Qué es?

Es un framework de desarrollo web de código abierto, mantenido por Google, que se utiliza para crear aplicaciones web de una sola página (SPA, por sus siglas en inglés). Fue lanzado en 2010 y está diseñado para facilitar el desarrollo y las pruebas de aplicaciones mediante la provisión de una arquitectura bien definida y de herramientas de desarrollo integradas.

Características

Data
Binding

Permite enlazar datos entre el modelo y la vista.

MVC

Implementa el patrón Modelo-Vista-Controlador.

Directivas

Extiende el HTML con nuevas funcionalidades.

Inyección
de Dependencias

Gestiona la creación y la vinculación de dependencias.

Testing

Diseñado para ser testeado fácilmente.

Instalación y configuración

Instalación

Node.js y npm:

Necesarios para administrar paquetes.

Instalación:

npm install -g @angular/cli

Configuración

Crear un nuevo proyecto:

ng new my-app

Navegar al directorio del proyecto:

cd my-app

Iniciar el servidor de desarrollo:

ng serve

Laravel

¿Qué es?

Es un framework de desarrollo web de código abierto escrito en PHP. Fue creado por Taylor Otwell y se lanzó por primera vez en junio de 2011. Laravel está diseñado para facilitar el desarrollo de aplicaciones web mediante una sintaxis elegante y expresiva.

Características

Artisan
CLI

Línea de comandos para tareas comunes.

ORM
(Eloquent)

Simplifica las interacciones con la base de datos.

Blade
Templating Engine

Sistema de plantillas para la vista.

Middleware

Gestiona las solicitudes HTTP.

Modularidad

Usa paquetes para extender funcionalidades.

Instalación y configuración

Instalación

Composer:

Administrador de dependencias de PHP.

Instalación:

composer create-project --prefer-dist laravel/laravel my-app

Configuración

Configurar el entorno:

.env

Generar la clave de aplicación:

php artisan key:generatep

Servidor de desarrollo:

php artisan serve

Node JS

¿Qué es?

Es un entorno de ejecución de JavaScript del lado del servidor. Fue desarrollado por Ryan Dahl y lanzado por primera vez en 2009. Node.js está construido sobre el motor V8 de Google Chrome, lo que le permite ejecutar código JavaScript de manera eficiente y rápida fuera de un navegador web.

Características

Asincronismo

Maneja operaciones I/O de manera asíncrona.

Event-Driven

Basado en eventos para mayor eficiencia.

NPM

Gran repositorio de paquetes y módulos.

Single-Threaded

Usa un solo hilo para manejar múltiples conexiones.

Instalación y configuración

Instalación

Descargar desde la página oficial:

Node.js

Comprobar instalación:

node -v y npm -v

Configuración

Crear un archivo:

package.json: npm init

Instalar paquetes necesarios:

npm install express --save

Crear un archivo para el servidor:

app.js

Spring

¿Qué es?

Es un framework de código abierto para el desarrollo de aplicaciones empresariales en Java. Fue creado por Rod Johnson y lanzado por primera vez en 2002. Spring proporciona una infraestructura integral para desarrollar aplicaciones robustas y de alto rendimiento, y es especialmente popular en el desarrollo de aplicaciones empresariales..

Características

Inyección
de Dependencias

Simplifica la gestión de componentes.

Modularidad

Compuesto por varios módulos (Spring Core, Spring MVC, etc.).

Aspect-Oriented
Programming (AOP)

Permite separar aspectos transversales.

Seguridad

Módulo Spring Security para gestionar la seguridad de la aplicación.

Acceso
a Datos

Módulo Spring Data para interacciones con bases de datos.

Instalación y configuración

Instalación

Java y Maven:

Necesarios para construir proyectos.

Spring Boot:

Herramienta para inicializar proyectos Spring fácilmente.

Configuración

Crear un proyecto con Spring Initializr:

Spring Initializr

Descomprimir y abrir el proyecto en un IDE:

(IntelliJ, Eclipse, etc.)

Configurar el archivo para ajustes específicos

application.properties
